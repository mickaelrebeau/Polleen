Aide

Retour à L'accueil

I. S'inscrire sur Polleen

Pour vous inscrire, un nom d'utilisateur, "Username", et un mot de passe, "Password", vous sera demandé.

II. Se connecter à votre profil

Pour vous connecter, il vous suffit de renseigner votre "Username" qui est votre nom d'utilisateur, ainsi que votre "Password" qui est votre mot de passe.

III. Fonctionnalités

a. Accueil

Les notifications concernant Polleen vous serons transmisent sous le bandeau "Notifications" et les actualités de Polleen et Pollen vous seront transmisent sous le bandeau "Actualités".

b. Répertoire

Depuis la page "Répertoire", cliquez sur le nom d'un de vos contacts afin de voir le détail de ce contact.

b.1. Ajouter un contact

Depuis la page "Répertoire", cliquez sur "Ajouter un contact", renseignez tous les champs obligatoires (marquer par une étoile) et cliquez sur "Valider". Vous avez ajouté un nouveau contact.

b.2. Modifier un contact

Depuis la page "Répertoire", cliquez sur le nom du contact que vous souhaitez modifier, cliquez sur "Mettre à jour" et modifier les informations souhaiter. Cliquez sur "Valider" afin de confirmer vos modifications.

b.3. Supprimer un contact

Depuis la page "Répertoire", cliquez sur le nom du contact que vous souhaitez supprimer, cliquez sur "Supprimer". Un message de confirmation apparaitra et cliquez sur "Supprimer" pour confirmer la suppression du contact.

b.4. Catégories

Depuis la page "Répertoire", cliquez sur "Catégories". Cliquez sur une des catégories afin de voir le détail de cette catégorie.

b.4.1. Ajouter une catégorie

Depuis la page "Répertoire", cliquez sur "Catégories". Cliquez sur "Créer une nouvelle catégorie". Complétez le nom de la catégorie et cliquez sur "Valider".

b.4.2. Modifier une catégorie

Depuis la page "Répertoire", cliquez sur "Catégories". Cliquez sur une des catégories afin de voir le détail de cette catégorie. Cliquez sur "Mettre à jour". Modifier le nom de la catégorie et cliquez sur "Valider".

c. Répertoire Public

Depuis la page "Répertoire Public", cliquez sur le nom d'un des contacts public afin de voir le détail de ce contact.

d. Documentation

Depuis la page "Documentation", cliquez sur "Voir la présentation" sur la présentation de votre choix afin de voir le détail de cette présentation.

IV. Réinitialiser son mot de passe

Depuis la page de connexion, cliquez sur "Mot de passe oublié?", renseignez votre adresse mail et cliquez sur "Valider". Vous recevrez un mail avec un lien, cliquez dessus et vous serez rediriger sur une nouvelle page afin de changer de mot de passe. Renseignez un nouveau mot de passe. Cliquez sur "Valider". Vous serez redirigier sur une page de confirmation, cliquez sur "Connexion". Vous pouvez maintenant vous connecter.
